# Task Manager (React + Vite)

A simple Task Manager app built with React and Vite.

## Features
- Create, read, update, delete tasks (CRUD)
- Mark tasks as complete
- Search + filter (all/active/completed)
- Saves tasks in LocalStorage (persists after refresh)

## Tech Stack
- React (Hooks)
- TypeScript
- Vite

## Run Locally
```bash
npm install
npm run dev
